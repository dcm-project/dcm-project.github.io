Welcome to the **Data Center Management (DCM)** documentation. Here you'll find
guides, architecture decisions, and technical references for building and
operating DCM-based infrastructure.

## Getting Started

DCM provides a hyperscaler-like cloud experience for enterprise on-premises and
sovereign cloud infrastructure. It uses an **API-first, technology-agnostic
framework** built on a declarative data model.
